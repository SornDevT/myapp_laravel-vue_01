ຕິດຕັ້ງ laravel: run ຄຳສັ່ງໃນ cmd
- composer global require laravel/installer

ສ້າງໂປຼເຈັກ Laravel 8 ໃໝ່ (example ຄືຊື່ໂປຼເຈັກຕັ້ງເອງ)
- laravel new example 

ເມື່ອເປີດໂປຼເຈັກຢູ່ VScode ແລ້ວຄ້ອງການ Compiled ແມ່ນໃຊ້ຄຳສັ່ງ
- npm run dev

ຕ້ອງການໃຫ້ Laravel run Compiled ທຸກຄັ້ງທີ່ມີການ ແກ້ໄຂ Save ຂໍ້ມູນໃຫ້ໃຊ້ຄຳສັ່ງ
- npm run watch

ຕ້ອງການ Run ໂປຼເຈັກໃຫ້ໃຊ້ຄຳສັ່ງ
- npm run serve

ຕ້ອງການ ຍຸດຄຳສັ່ງທີ່ກຳລັງ Run ຢູ່ແມ່ນໃຫ້ກົດ: Ctrl + C 1 ຫຼື 2 ຄືຊື່ໂປຼເຈັກຕັ້ງເອງ

ການຕິດຕັ້ງ Packet ຫຼື Library ຂອງ Vue ໃຫ້ພິມ: npm install ຫຼືແບບຫຍໍ້ npm i ແລ້ວຕາມດ້ວຍ ຊື່ Packet ຫຼື Library
ແລະ ຖ້າຕ້ອງການ ຖອນຕິດຕັ້ງ ໃຫ້ພິມ npm uninstall ແລ້ວຕາມດ້ວຍຊື່ Packet
